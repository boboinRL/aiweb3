## Privacy & Security Notice

This repository may be public. Do **not** commit API keys, tokens, passwords, private keys, seed phrases, wallet mnemonics, non-public contact info, private meeting links, private chats, personal data of others, exchange credentials, or sensitive trading account screenshots.

## Directory Map

- `profile.md` — learner profile and goals
- `learning-plan.md` — personal learning roadmap
- `daily/` — daily notes and check-in drafts
- `tasks/` — task notes, submissions, proof-of-work
- `experiments/` — small research/code/trading experiments
- `handbook-feedback/` — feedback items for Handbook improvement
- `hackathon/` — hackathon/project ideas and prototypes
- `submissions/` — submitted proof links and records
- `templates/` — reusable note templates

## Operating Principle

Build a visible proof-of-work trail: learn concepts, run small experiments, write notes, identify feedback, and connect learning to practical trading/quant/Web3 workflows.
