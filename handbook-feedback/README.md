# Handbook Feedback

Use this directory to record feedback that can improve the AI × Web3 School Handbook.

Each feedback item should include:

- Handbook page link
- Issue type: typo / unclear concept / outdated resource / missing example / structure suggestion
- Problem description
- Suggested change
- Source: daily note, experiment, discussion, or task

## Workflow

1. During learning, capture friction immediately.
2. Create a feedback file from `templates/handbook-feedback.md`.
3. Keep it public, specific, and actionable.
4. If submitting upstream later, link the issue/PR/submission back here.
